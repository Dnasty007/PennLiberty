Save future Triplex photos in this folder so the app can load them from `/public/listings/1704-w-diamond/`.

Recommended filenames:
- `cover.jpg`
- `gallery-1.jpg`
- `gallery-2.jpg`
- `gallery-3.jpg`
- `gallery-4.jpg`

These map cleanly to URLs like:
- `/listings/1704-w-diamond/cover.jpg`
- `/listings/1704-w-diamond/gallery-1.jpg`

If you add different filenames later, update the `image` and `gallery` fields in `src/lib/data.ts`.
